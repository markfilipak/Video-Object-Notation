Six Element Objects & 12 Stream Objects —— a table of element object & stream object characteristics.
The following objects each possess physical characteristics that uniquely differentiate them. They are the object-oriented equivalents of datatypes but can also possess uniquely suited properties and methods.

Half resolution image streams & field streamsFull resolution image streams & frame streams
s (scan): the native scan element.
• resolution: half.
• pixels: consecutive, left to right, top to bottom.


sp (scanpic): a consecutive pair of scans, weaved — a
pseudo picture.
• resolution: full.
• shows combing in every image containing motion.

sps (scans-per-second): a stream of s elements.
sps streams have the following intrinsic properties:
• resolution: half.
• pipe rate: sps.
Method sps.spps weaves to produce a new spps stream.
Method sps.] encodes to produce a new sfps stream.
sfps (scan-frames-per-second): field encoded sps.
sfps streams have the following intrinsic properties:
• resolution: half (per field).
• pipe rate: sfps = sps/2.
Method sfps.[ decodes to produce a new sps stream.

spps (scanpics-per-second): a stream of sp elements.
spps streams have the following intrinsic properties:
• resolution: full.
• pipe rate: spps = sps/2.
• shows combing in every image containing motion.
Method spps.sps unweaves to produce a new sps stream.
Method spps.] encodes to produce a new spfps stream.
spfps (scanpic-frames-per-second): frame encoded spps.
spfps streams have the following intrinsic properties:
• resolution: full.
• pipe rate: spfps = spps.
• shows combing in every frame containing motion.
Method spfps.[ decodes to produce a new spps stream.

h (halfpics): two concurrent half 'pictures' as
unweaved from a single 'picture'.
• resolution: half.

p (picture): the native 'picture' element.
• resolution: full.
• pixels: simultaneous.

hps (halfpics-per-second): a stream of h elements.
hps streams have the following intrinsic properties:
• resolution: half.
• pipe rate: hps = pps*2 [note2].
• hps weaved together is pps.
Method hps.pps weaves to produce a new pps stream.
Method hps.(AaBbCcDd(AaBbBcCdDd)) uses stride-match
to produce a new tps stream, 'NTSC' telecine in this
case. [note3]
Method hps.] encodes to produce a new hfps stream.
hfps (halfpic-frames-per-second): field encoded hps.
hfps streams have the following intrinsic properties:
• resolution: half (per field).
• pipe rate: hfps = hps/2.
Method hfps.[ decodes to produce a new hps stream.

pps (pictures-per-second): consecutive p elements
— called progressive by some authorities.
pps streams have the following intrinsic properties:
• resolution: full.
• pipe rate: pps.
Method pps.hps unweaves to produce a new hps stream.
Method pps.] encodes to produce a new pfps stream.
pfps (picture-frames-per-second): frame encoded pps.
pfps streams have the following intrinsic properties:
• resolution: full.
• pipe rate: pfps = pps.
Method pfps.[ decodes to produce a new pps stream.

th (telehalfpics): two non-concurrent half 'pictures'
as unweaved from two consecutive 'pictures'.
• resolution: half.

tp (telepics): a th element, weaved — a pseudo picture.
• resolution: full.
• shows combing in every image containing motion.

tps (telecines-per-second): a stream of h elements
combined with strategically placed th elements.
tps streams have the following intrinsic properties:
• resolution: half.
• pipe rate: tps = hps*(h_count+th_count)/(h_count+1).
Method tps.tpps weaves to produce a new tpps stream.
Method tps.(AaBbBcCdDd(AaBbCcDd)) uses stride-match
to produce a new hps stream, 'NTSC' detelecine in
this case. [note3]
Method tps.] encodes to produce a new tfps stream.
tfps (telecine-frames-per-second): field encoded tps.
tfps streams have the following intrinsic properties:
• resolution: half (per field).
• pipe rate: tfps = tps/2.
Method tfps.[ decodes to produce a new tps stream.

tpps (telepics-per-second): a stream of p elements
combined with strategically placed tp elements.
tpps streams have the following intrinsic properties:
• resolution: full.
• pipe rate: tpps = pps*(p_count+tp_count)/(p_count+1).
• shows combing in every tp image containing motion.
Method tpps.tps unweaves to produce a new tps stream.
Method tpps.] encodes to produce a new tpfps stream.
tpfps (telepic-frames-per-second): frame encoded tpps.
tpfps streams have the following intrinsic properties:
• resolution: full.
• pipe rate: tpfps = tpps.
• shows combing in every tp frame containing motion.
Method tpfps.[ decodes to produce a new tpps stream.

[note1] Here is a mapping of the stream-to-stream production methods: sfps———sps———spps———spfps

                                                                      hfps———hps———pps————pfps
                                                                              |
                                                                      tfps———tps———tpps———tpfps
